What Changes at Each Scale Stage

🌱 SEED — 0 to 1K MAU

What you need

  • Stack: Railway + Supabase + Modal. ~$7 infra/app.
  • AI API: Modal free $30 credit + Anthropic free tier. Near-zero start.
  • DB: Single Supabase project handles all 10 apps.
  • Model routing: Gemini Flash for all free users. Haiku for paid only.
  • Focus: Ship fast. Don't over-engineer. Every rupee goes to users, not infra.
🚀 GROWTH — 1K to 10K MAU

What changes

  • DB: Upgrade Supabase compute to Small ($15/mo). Separate projects per high-traffic app.
  • Caching: Redis on Railway ($8/mo) cuts API calls 30–40%.
  • API cost: Now biggest line item. Implement smart model routing.
  • GPU: Modal concurrency at 10 — upgrade to Team plan if Image/Voice hitting limits.
  • Credits: Apply AWS Activate + GCP now. This is exactly the stage they target.
⚡ SCALE — 10K to 100K MAU

What you must change

  • GPU: Image/Voice → AWS SageMaker or GCP Cloud Run reserved. 40% cheaper than Modal at volume.
  • DB: Supabase → RDS/Cloud SQL or Supabase Business. pgvector → Pinecone at 10M+ vectors.
  • API pricing: Negotiate enterprise with Anthropic at $50K+/mo spend. ~20–30% discount.
  • Credits: ₹3 crore in GCP/AWS credits covers all infra for 12–18 months.
  • Region: Add Mumbai replica — Railway → GCP Cloud Run Mumbai for Indian users.
